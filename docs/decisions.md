## 2026-07-07 — 팔레트 전환 "자기와 물" (warm modern 폐기)

커피 브라운 단일 색조 팔레트(아이보리 + 에스프레소 + 오커)는 화면 전체 hue 폭이 10~15°에 갇혀 덥고 텁텁했고, 크림+오커 조합 자체가 시대 기본값이 되어 올드하게 읽힘. 색의 주제를 원두에서 **앱이 실제로 다루는 것(물·온도·자기 드리퍼)**으로 재해석 — 본화이트 자기 뉴트럴 + 청먹 잉크 + 심수 청록 액센트로 전환. 프로토타입 3안 비교 후 "안 2 — 자기와 물" 채택 (액센트 hue -5°·채도 ×1.10 조정 반영). 브루잉 lead-in 연출이 "화면이 데워짐"에서 "물이 스밈"으로 의미 일치. 값 단일 출처는 `design-tokens.md`, 원칙 개정은 `brand.md` § 비주얼 언어. 순수 무채색 그레이는 계속 금지(언더톤 필수) — "병원 느낌" 회피 원칙은 유지.
